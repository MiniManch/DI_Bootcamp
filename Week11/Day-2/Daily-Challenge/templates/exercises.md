<h4>Instructions</h4>

<ol>

<li>Create a function that determines whether an argument is omnipresent for a given array.<br />

<pre><code>[[3, 4], [8, 3, 2], [3], [9, 3], [5, 3], [4, 3]]

</code></pre>
